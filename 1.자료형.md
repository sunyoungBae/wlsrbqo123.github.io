# 자료형
## 숫자형
* 정수형 : 양의 정수(123), 음의 정수(-123), 0
* 실수형
  - 기본
  ```
  >>> a = 4.2
  >>> b = -4.2
  ```
  - 컴퓨터식 지수 표현 방식
  ```
  >>> a = 4.24E10  # 4.24 * 10의 10승
  >>> b = 4.24e-10 # 4.24 * 10의 -10승
  ```
* 8, 16진수
  - 8진수 : 0o 또는 0O(숫자 0 + 알파벳 o의 소문자 또는 대문자)로 시작
  ```
  >>> a = 0o177
  ```
  - 16진수 : 0x로 시작
  ```
  >>> a = 0x8ac
  ```
* 연산
  - 사칙연산 : +, -, *, /
  - x의 y제곱 : ** 연산자
  ```
  >>> 3 ** 4
  81
  ```
  - 나눗셈 후 나머지를 반환 : % 연산자
  ```
  >>> 7 % 4
  3
  ```
  - 나눗셈 후 몫을 반환 : // 연산자
  ```
  >>> 7 // 4
  1
  ```
